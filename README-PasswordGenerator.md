# Password Generator

A simple Java Swing application to generate secure passwords up to **2048 characters**.

## Features
- Choose character sets:
  - Uppercase (A-Z)  
  - Lowercase (a-z)  
  - Numbers (0-9)  
  - Symbols (!@#$%&*./?)  
- Adjustable password length (8â€“2048)  
- Copy generated password to clipboard with one click  

## Requirements
- Java 17 or higher
